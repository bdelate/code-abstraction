1. Overview

Data to Site allows users to emded cards into their website that are powered by spreadsheet data.

When they add / remove data from their spreadsheet, the cards on their website are automatically updated. This allows users without programming knowledge to customize their cards and keep their website up to date.

A few key features include:

  • Card customization using a visual designer.
  • No coding required by the user.
  • Can be powered by any csv data source.
  • Data on the users site updates automatically when spreadsheet data is added / removed.
  • Card filter / search
