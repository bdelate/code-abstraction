Diversify Portfolio

Diversify Portfolio allows investors to analyze their stock portfolios and look for opportunities using several interactive tools.

A few key features include:

  • 5 years of data on over 7 500 stocks, updated daily.
  • Portfolio level backtesting and analysis.
  • Interactive charts and visualisation tools.
  • Stocks screens and opportunity finders.
  • PayPal integration.
  • MailChimp integration.

Kanban Board

This is a Kanban board implemented as a SPA using React and Redux on the front end with Django Rest Framework on the back end.

A few key features include:

  • Multiple boards per user.
  • Column and card (task) creation.
  • Dragging of cards between columns.
  • Dragging of cards within a column to sort / organise them.

Who Knows - Q&A

Who Knows is a question and answer site similar to Stack Overflow or Quora.

It includes all the core functionality you would expect from a Q&A site such as:

  • Voting for questions, answers or comments
  • Accepting answers
  • Tagging and filtering
  • Searching
